## DEMO PROJECT

### Node команды

```sh
# установка всех записимостей из файла package.json, если такие есть
npm install

# запустить сервер для удобной разработки
npm run dev

# собрать все скрипты в один
npm run build

# выполнить джаваскрипт файл из Node.js
node путь_к_файлу.js
```

### Git команды

```sh
# Загрузить всю инфу с репозитория на ГитХабе по оказанной ветке
git pull origin название_ветки_которая_нужна

# Добавить все файлы, чтобы Гит следил за их изменениями
git add -A

# Статус текущих файлов
git status

# какие есть изменения в файлах
git diff

# создать новую ветку, которая отколится от текущей
git checkout -b 'имя_ветки'

# Сохранить локально изменения во всех файлах с выбранным сообщением
git commit -a -m 'сообщение для сохранения'

# Загрузить всю инфу в репозиторий
git push origin 'название_текущей_ветки'
```
